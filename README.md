# edinburgh.dog

## Install Dependancies

Requires: NodeJS, NPM, Ruby

### Global Dependancies
`npm install -g bower grunt`
`gem install bundler jekyll`

### Local Dependancies
`npm install .`
`bundle install -g grunt`
`bundle install .`
`bower install`

## Build
`bundle exec rake build`

## Develop
`grunt serve`
